Detailed Description

template<typename T, typename VectorType>
class esp32_psram::TypedRingBuffer< T, VectorType >

A generic ring buffer implementation for any data type.

Template Parameters
TThe data type to store in the buffer
VectorTypeThe vector type to use as underlying storage

This class implements a ring buffer (circular buffer) that uses a vector container for storage. Unlike the Stream-based RingBuffer, this can store any data type.
